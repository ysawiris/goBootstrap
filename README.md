# GoBootstrap

    A golang web scrapper that collects all the code snippets for each Boostap Component

# Usage

1. Clone the repo
   \$ git clone https://github.com/ysawiris/goBootstrap.git
2. Build and run the code
   \$ go build && ./makescrapper
3. To see file
   \$ open test.txt
